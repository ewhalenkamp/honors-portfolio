Project Description

This project lists each and every Pokémon instance until the start of the current Generation, Gen VIII. By clicking on a Pokémon's image, the site user can view individual and detailed information about that Poké, including their stats, moves, abilities, and more.

This Pokédex version is a successor to a class project I completed just under one year ago for a semester project. As it was an introductory programming course and I didn't know how to use web technologies at the time, I implemented this project by compiling nearly a thousand HTML files for each Pokémon instance with a Java program. The same database I used in that version of the project came into play in creating an API with Springboot. Lastly, I created a Django backend to serve the original version's HTML with context instead of hard-written values.
