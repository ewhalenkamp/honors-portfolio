Project Description

Multi Music Player accesses multiple YouTube APIs, allowing users to search for music and add it to their queue to be played in-app. It functions as a playlist off the YouTube website itself, and has as-of-yet proven to be an ad-less alternative to regular YouTube playlists.

This application was programmed in sprints over a short duration of time. Check out the documentation under the GitHub link for more information.
